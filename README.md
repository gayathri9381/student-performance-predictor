# Student Performance Prediction System

##  Overview
This project predicts student academic performance using Machine Learning techniques.  
The system analyzes student-related factors such as attendance, study hours, assignments, and internal marks to identify students who may need academic support.

The project helps educational institutions improve student success through early prediction and intervention.

---

##  Features
- Predicts student academic performance
- Data preprocessing and cleaning
- Student data analysis and visualization
- Machine Learning model training
- Accuracy evaluation
- Easy-to-use prediction system
